# A股做T分析工具 - 差异化竞争分析

## 竞品：ZhuLinsen/daily_stock_analysis (10.1k stars)

### 它的优势
- 功能全面：技术分析 + 基本面 + 新闻情绪
- Web UI 可视化
- 多模型支持（Gemini/DeepSeek）
- 多渠道推送
- 社区活跃

### 它的劣势（我们的机会）
1. **太泛了** - 什么都有但什么都不精，做T用户只需要买卖点
2. **依赖 LLM API** - 需要 Gemini API Key，有成本
3. **没有做T收益率追踪** - 不记录历史建议的准确率
4. **不是纯A股** - 也做美股港股，不够专注
5. **没有具体的操作触发条件** - 只说"买入""卖出"，不说"当价格跌破X时买入"

## 我们的差异化

### 1. 聚焦做T（做窄做深）
- 只做一件事：帮散户做T
- 不做基本面分析、新闻情绪、投资组合等
- 标语："做T，只需要这一个工具"

### 2. 零外部依赖（开箱即用）
- 不需要任何 API Key
- pip install 就能用
- 纯 Python 标准库 + 三个常用包

### 3. 做T收益追踪（独有功能）
- 记录每天的建议买卖价
- 自动对比实际收盘价，计算建议准确率
- 周报/月报：本月做T建议收益率

### 4. 精确触发条件（不是模糊区间）
- 不说"买入区间 16.20-16.35"
- 而说"当 5 分钟级别跌破 16.28 且成交量低于 5000 手时买入"
- 加入场时机判断（开盘30分钟内不操作等）

### 5. 个人版 vs 专业版（变现路径）
- 个人版：免费开源，基础功能
- 专业版（付费）：自动盯盘、实时推送、多股组合优化
