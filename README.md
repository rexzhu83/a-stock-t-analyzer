## 🚀 快速开始
### 安装

```bash
git clone https://github.com/rexzhu83/a-stock-t-analyzer.git
cd a-stock-t-analyzer
pip install -r requirements.txt
```

### 使用

```bash
# 分析单只股票
python analyzer.py --stock sh600791

# 分析多只股票
python analyzer.py --stock sh600791 sh603268 hk06869

# 使用更多历史数据（默认60天）
python analyzer.py --stock sh600791 --days 120
```

## 📈 做T收益追踪

内置 `t_tracker.py` 模块，可记录每日建议买卖价并自动对比实际成交情况，考核做T准确率和收益。

```python
from t_tracker import TTracker

tracker = TTracker()

# 添加做T记录
tracker.add_record(
    stock_code="sh600791",
    date="2026-03-17",
    buy_price=7.98,        # 建议买入价
    sell_price=8.50,       # 建议卖出价
    actual_low=7.94,       # 当日实际最低
    actual_high=8.47,      # 当日实际最高
    actual_close=8.29,     # 当日收盘价
)

# 查看统计报告
print(tracker.get_report())
# → 买入命中率、卖出命中率、完美做T次数、月度预估收益

# 导出 CSV
tracker.export_csv("data/my_records.csv")
```

**追踪指标：**

| 指标 | 说明 |
|------|------|
| 💯 买入/卖出命中率 | 建议价格是否被实际行情触发 |
| ✅ 完美做T率 | 双向都命中的比例 |
| 💰 月度预估收益 | 按1万元本金算 |

## 🔬 回测验证

内置 `backtest.py` 模块，用历史数据自动回测做T买卖信号的准确率，帮助你了解信号的真实表现。

### 验证逻辑

- **买入信号**：触发后 N 日内价格不再下跌 ↑ ✅ 有效（说明选对了时机）
- **卖出信号**：触发后 N 日内价格不再上涨 ↑ ✅ 有效（说明卖在了高点附近）

### 使用

```bash
# 对单只股票进行回测（默认回测60天数据，信号触发后看3天）
python backtest.py --stock sh600791

# 调整回测参数
python backtest.py --stock sh600791 --days 120 --forward 5

# 输出 JSON 格式
python backtest.py --stock sh600791 --json
```

## 📐 技术指标说明

| 指标 | 说明 | 买入信号 | 卖出信号 |
|------|------|---------|---------|
| **MA均线** | 移动平均线 | 多头排列 (5>10>20) | 空头排列 (5<10<20) |
| **MACD** | 趋势指标 | DIF 上穿 DEA（金叉） | DIF 下穿 DEA（死叉） |
| **KDJ** | 随机指标 | J < 20（超卖） | J > 80（超买） |
| **RSI** | 相对强弱指数 | RSI < 30（超卖） | RSI > 70（超买） |
| **布林带** | 波动率指标 | 触及下轨 | 触及上轨 |

## 💡 为什么做这个工具？

做T（日内高抛低吸）是A股散户最实用的赚钱策略之一。但手动分析支撑位、阻力位、技术指标太耗时。这个工具自动化了整个分析流程，每天花30秒就能得到专业的做T建议。

### 与其他工具的区别

- **专注做T** — 不做基本面、新闻情绪，只专注买卖点
- **零外部依赖** — 不需要 API Key，`pip install` 就能用
- **收益追踪** — 独有功能，记录历史建议的准确率
- **精确触发条件** — 告诉你具体价格，不是模糊区间
